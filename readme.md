Bot Commands

Mentions Commands
* !n mentions [limit]
  Shows your most recent mentions. Optional limit sets how many messages to display (default 10).

* !n mentionsof <user> [limit] (Owner only)
  Shows the most recent mentions of the specified user. Optional limit sets how many messages to display (default 10).

------------------------------------------------------------------------------------
Subscriber Management Commands (Owner only)
* !n addsub <user>
  Adds a new subscriber.

* !n removesub <user>
  Removes a subscriber.

* !n enablesub <user>
  Enables a previously disabled subscriber.

* !n disablesub <user>
  Disables a subscriber without removing them.

* !n subscribers
  Lists all subscribers and their status (enabled/disabled).
